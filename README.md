# Procurement Analytics Dashboard - Excel

An interactive Excel dashboard built for SAE BAJA Team Samanvaya Racing procurement tracking.

## Project Objective
To analyze procurement spending, vendor performance, order status, and subsystem-wise purchase patterns using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- VBA Macros

## Key Features
- Interactive subsystem slicer
- KPI cards for total spend, vendors, quantity, and pending orders
- Vendor-wise spend analysis
- City/year-wise spend trend
- Payment status analysis
- VBA-based selective slicer connection controls

## Key Insights
1) Procurement spending is concentrated in a few key subsystems.
2) Vendor dependency is visible across major purchase categories.
3) A significant share of orders remains pending, requiring follow-up.
